# Depthful - Features & Tips Guide

## Overview

Depthful is a thoughtful journaling app designed to help you capture, organize, and reflect on your thoughts, ideas, and experiences. With support for text, voice recordings, photos, and rich markdown formatting, Depthful provides a flexible and powerful platform for personal reflection.

---

## Core Features

### 1. Thought Management

**Creating Thoughts**
- Tap the "+" button to create a new thought
- Write freely with support for plain text or markdown formatting
- Thoughts are automatically saved as you type
- Empty thoughts are automatically discarded to keep your journal clean

**Editing Thoughts**
- Tap any thought to open and edit it
- Changes are saved automatically
- View creation and last updated timestamps
- Mark thoughts as favorites with the star icon

**Organizing Thoughts**
- Sort by: Newest First, Oldest First, Recently Viewed, Least Recently Viewed, A to Z, or Z to A
- Favorites always appear at the top regardless of sort order
- Search through all your thoughts using the search bar
- Filter by tags to find related thoughts quickly

### 2. Rich Text Formatting

**Markdown Support**
Depthful supports markdown formatting for rich text:

- **Bold**: `**text**` or use the formatting toolbar
- *Italic*: `*text*` or use the formatting toolbar
- _Underline_: `_text_` or use the formatting toolbar
- `Code`: `` `text` `` or use the formatting toolbar
- ~Strikethrough~: `~text~` or use the formatting toolbar
- Links: `[Link Text](https://example.com)` or use the formatting toolbar

**Formatting Toolbar**
- Access quick formatting buttons while editing
- Preview your formatted text before saving
- Toggle between edit and preview modes

### 3. Tags System

**Default Tags**
Depthful comes with 18 pre-configured tags:
- Advice, Daydream, Dream, Experience
- Goal, Gratitude, Idea, Inspiration
- Journal, Learning, Memory, Milestone
- Question, Quote, Reflection, Regret
- Story, Thought

Each default tag has a unique color for easy visual identification.

**Custom Tags**
- Create unlimited custom tags with your own names and colors
- Edit existing custom tags (name and color)
- Delete custom tags you no longer need
- Recently used tags appear at the top for quick access

**Multi-Tag Support**
- Apply multiple tags to a single thought
- Filter by one or more tags simultaneously
- View all thoughts with specific tag combinations
- See tag counts in the analytics view

**Tag Management**
- Access tag selection from the main view or while editing a thought
- Search through tags to find the one you need
- View selected tags at the top of the tag selection screen
- Clear all selected tags with one tap

### 4. Voice Recordings

**Recording Features**
- Record up to 5 minutes of audio per recording
- Real-time waveform visualization while recording
- Pause and resume recording
- See elapsed time and progress bar
- Automatic transcription (requires Speech Recognition permission)

**Transcription**
- Automatic speech-to-text conversion
- Copy transcription to clipboard with one tap
- Transcriptions are saved with the recording
- Works offline once permission is granted

**Managing Recordings**
- Recordings are automatically named with date and time
- Attach recordings to specific thoughts
- View all recordings in the recordings gallery
- Play, pause, and delete recordings
- See recording duration and creation date

### 5. Photo Attachments

**Adding Photos**
- Attach multiple photos to any thought
- Photos are automatically compressed for optimal storage
- View photos in a gallery layout
- Tap photos to view full-screen

**Photo Management**
- Delete individual photos from a thought
- Photos are stored securely in Core Data
- Export thoughts with photos included
- View photo count indicators on thought cards

### 6. Analytics & Insights

**Overview Statistics**
- Total number of thoughts

**Activity Tracking**
- View activity over different time ranges (7 days, 30 days, previous year, all time)

**Writing Streaks**
- Current streak: consecutive days with at least one thought

**Tag Analytics**

**Additional Insights**

## Tips & Best Practices

### Getting Started

1. **Start Simple**: Begin by writing a few thoughts without worrying about tags or formatting
2. **Explore Tags**: Try the default tags to see which ones resonate with your thinking style
3. **Create Custom Tags**: Add tags that match your specific needs (e.g., "Work", "Family", "Projects")
4. **Use Voice Recording**: Try voice notes when typing isn't convenient

### Organization Tips

1. **Use Multiple Tags**: Apply 2-3 relevant tags to make thoughts easier to find later
2. **Favorite Important Thoughts**: Star thoughts you want to reference frequently
3. **Regular Review**: Use the analytics view to see patterns in your thinking
4. **Search Effectively**: Use the search bar to find specific words or phrases across all thoughts

### Writing Tips

1. **Write Freely**: Don't worry about perfect grammar or structure
2. **Use Markdown**: Format important points with bold or italic for emphasis
3. **Add Context**: Include dates, locations, or people in your thoughts for future reference
4. **Attach Media**: Add photos or voice notes to capture moments more completely

### Privacy & Security

1. **Local Storage**: All your thoughts are stored locally on your device
2. **No Cloud Sync**: Your data never leaves your device (unless you export it)
3. **Secure Storage**: Data is stored in Core Data with iOS security protections
4. **Export Backups**: Regularly export your thoughts as a backup

### Performance Tips

1. **Photo Compression**: Photos are automatically compressed to save space
2. **Recording Limits**: 5-minute recording limit helps manage storage
3. **Regular Exports**: Export and archive old thoughts to keep the app responsive
4. **Delete Unused Tags**: Remove custom tags you no longer use

### Workflow Suggestions

**Daily Journaling**
- Create a "Daily" custom tag
- Write at the same time each day
- Track your streak in Analytics
- Review weekly patterns

**Idea Capture**
- Use the "Idea" tag for quick captures
- Add voice notes when inspiration strikes
- Review ideas weekly and expand on promising ones
- Use favorites to mark ideas worth pursuing

**Gratitude Practice**
- Use the "Gratitude" tag
- Write 3 things daily
- Review monthly to see positive patterns
- Export annually for reflection

**Project Notes**
- Create custom tags for each project
- Use multiple tags (e.g., "Work" + "Project Name")
- Attach relevant photos or voice notes
- Search by project tag to see all related thoughts

**Dream Journal**
- Use the "Dream" tag
- Write immediately upon waking
- Include as much detail as possible
- Review patterns in Analytics

## Troubleshooting

### Voice Recording Issues
- **No Transcription**: Check Settings > Privacy > Speech Recognition
- **Recording Fails**: Ensure microphone permission is granted
- **Poor Quality**: Record in a quiet environment

### Photo Issues
- **Can't Add Photos**: Check Settings > Privacy > Photos permission
- **Photos Not Showing**: Toggle "Show Photos" in display options
- **Large File Size**: Photos are automatically compressed

### Performance Issues
- **App Slow**: Try exporting and archiving old thoughts
- **Search Slow**: Reduce the number of thoughts or use more specific search terms
- **Crashes**: Ensure you're running the latest version

### Data Issues
- **Thoughts Not Saving**: Check available storage space
- **Tags Not Appearing**: Reload the tag list by navigating away and back
- **Export Fails**: Ensure you have write permissions for the destination
